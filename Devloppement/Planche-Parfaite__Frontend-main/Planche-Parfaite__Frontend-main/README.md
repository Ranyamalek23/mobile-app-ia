# Planche Parfaite

## Description
**Planche Parfaite** est une application mobile Android développée en Kotlin. Elle aide les utilisateurs à exécuter correctement l’exercice de la planche en utilisant des technologies de reconnaissance du corps humain. L’application analyse la posture en temps réel, chronomètre la durée de l’exercice, et fournit des retours visuels et sonores pour corriger la posture si besoin. 

L’objectif à long terme est de proposer une plateforme évolutive qui inclut d'autres exercices fonctionnels comme les pompes et d’autres mouvements de callisthénie.

## Fonctionnalités
- **Reconnaissance de posture :** Analyse en temps réel pour valider la posture correcte.
- **Chronométrage :** Suivi du temps pendant lequel la posture correcte est maintenue.
- **Feedback utilisateur :** Alertes visuelles et sonores en cas de posture incorrecte.
- **Espace personnel :** Connexion/inscription pour accéder à un tableau de bord avec historique et statistiques.
- **Historique :** Enregistrement des sessions d'exercices avec analyse des performances.
- **Paramètres personnalisables :** Notifications, unités de temps, et gestion des préférences utilisateur.

## Technologies Utilisées
- **Langage :** Kotlin (Android)
- **IDE :** Android Studio
- **Base de données :** MySQL
- **Backend :** API REST développée avec Node.js
- **Machine Learning :** TensorFlow Lite pour la reconnaissance de posture
- **Gestion du code :** GitHub

## Installation
1. Clonez le dépôt :
   ```bash
   git clone https://github.com/votre-repo/planche-parfaite.git
   ```
2. Importez le projet dans Android Studio.
3. Configurez la connexion avec le backend et la base de données MySQL.
4. Exécutez l’application sur un émulateur ou un appareil Android.
